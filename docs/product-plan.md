# JARVIS Stock War Room

Standalone App Store product for US equity scoring and AI research.

## Product Core

- Daily US market war room
- JARVIS Score for stocks and ETFs
- Ticker input analysis
- Watchlist and score tracking
- AI research summaries through a backend API

## Architecture

- Mobile app: Expo React Native
- Scoring engine: existing Python JARVIS project
- Backend API: FastAPI
- Database: Supabase/Postgres
- AI provider: OpenRouter, server side only
- Payments: RevenueCat

## First Release

- War Room home
- Top rated list
- Ticker analysis screen
- Risk disclaimer
- Mock fallback data until backend is connected

## App Store Compliance

JARVIS is positioned as a market education and research tool. It must not promise profit, give personalized financial advice, or route brokerage orders.
